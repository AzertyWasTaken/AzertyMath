Square number

Definition

The square sequence is a sequence where the nth number equal n multiplied by itself.

The \(10\) first square numbers are \(1, 4, 9, 16, 25, 36, 49, 64, 81, 100 \).

A number is square if and only if it have a odd number of divisors.

\(36\) is the smallest number that is both square and triangular.
