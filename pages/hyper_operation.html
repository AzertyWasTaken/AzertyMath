Hyper operation

Definition

Hyper operations is an extension of addition, multiplication and exponentiation.

Each level of operation is the repeatition of the previous level.

The level after exponentiation is called tetration.

Notation

These operations are denoted with the \(\uparrow \) symbol.

A single \(\uparrow \) represent exponentiation.

Two arrows \(\uparrow \uparrow \) represent tetration.

For each level of operation, one more \(\uparrow \) is added.

If there are many arrows, \(\uparrow^n\) can be used to describe \(n\) arrows in a row.

Properties

  • \(2 ↑^n 2 = 4\)
  • \(a ↑^n 1 = a\)
  • \(a ↑^n 0 = a\)
  • \(1 ↑^n b = 1\)
