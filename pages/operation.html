Operation

Definition

An operation is just an action you do with numbers to get a result.

The four basic ones are:

  • Addition \((+)\) → putting numbers together.
  • Subtraction \((-)\) → taking numbers away.
  • Multiplication \((\times)\) → adding the same number multiple times.
  • Division \((\div)\) → splitting into equal parts.

More complex operations exist, like exponentiation or hyperoperations.

Properties

An operation is commutative if changing the order of its numbers does not change the result.

The identity of an operation is a number \(n\) such as any number \(m\) and \(n\) equal \(m\).

An operation is unary if it takes a single value.

An operation is binary if it takes two values.
