Divisor

Definition

A divisor is a number that can divide another number evenly (with nothing left over).

They are like the “building blocks” that fit perfectly into a number.

An number \(n\) is divisible by a number \(m\) if \(m \div n\) have no remainder.

Example

  • \(3\) is a divisor of \(12\) because \(12 \div 3 = 4\) with no remainder.
  • \(5\) is not a divisor of \(12\) because \(12 \div 5 = \frac{12}{5}\) — the result has 2 left over.

The divisor count of the first ten numbers are \(1, 2, 2, 3, 2, 4, 2, 4, 3, 4\).

Properties

  • The fraction of an integer \(n\) and an integer not divisible by \(n\) result in a rational number.
  • An integer \(n\) is divisible by an integer \(m\) if \(n \div m\) have no remainder.
  • Every number is divisible by \(1\) and itself.
  • A number have a odd number of divisors if and only if it is square.
  • If a number \(n\) is a divisor of a number \(m\), \(m \div n \) is also a divisor of \(m\).
