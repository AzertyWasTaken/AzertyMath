Division

Definition

Division is the way we split something into equal parts or see how many times one number fits into another.

This is one of the four basic operations.

Division is denoted with the \(\div \) symbol.

The result of division is called the fraction.

Example

If you have \(12\) cookies üç™ and 3 friends, and you want to share them equally, each friend gets \(3\) cookies.

That is \(12 \div 3 = 4\).

Properties

  • Division is not commutative:

    • \(a \div b \neq b \div a\)

  • The identity of division is \(1\):

    • \(n \div 1 = n\)

  • 0 divided by any number equal \(0\):

    • \(0 \div n = 0\)

  • Any number (except \(0\)) divided by itself equal \(1\):

    • \(n \div n = 1\)

  • If \(a\) or \(b\) is negative and the other number is positive:

    • \(a \div b \lt 0\)

  • If \(a\) and \(b\) are both negative:

    • \(a \div b \gt 0\).
