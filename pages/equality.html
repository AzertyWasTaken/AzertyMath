Equality

Definition

Equality is a relationship between two quantities, stating that they have the same value.

Equality is denoted with the \(=\) symbol.

Example

\(11\) is the same number than the result of \(7 + 4\).

That is \(7 + 4 = 11\).

Inequality

Inequality is denoted with the \(\neq \) symbol.

If a number is less than another number, the inequality can be denoted with the \(\lt \) symbol.

If a number is greater than another number, the inequality can be denoted with the \(\gt \) symbol.
