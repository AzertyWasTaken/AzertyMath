Multiplication

Definition

Multiplication is a quick way of doing repeated addition.

This is one of the four basic operations.

Multiplication is denoted with the \(\times \) symbol.

The result of multiplication is called the product.

Properties

  • Multiplication is commutative:

    • \(a \times b = b \times a\)

  • The identity of multiplication is 1:

    • \(n \times 1 = n\)

  • Any number multiplied by \(0\) equal 0:

    • \(n \times 0 = 0\)

  • If \(a\) or \(b\) is negative and the other number is positive:

    • \(a \times b \lt 0\)

  • If \(a\) and \(b\) are both negative:

    • \(a \times b \gt 0\)

  • \(n \times (a + b) = n \times a + n \times b\)
