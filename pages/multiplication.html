Multiplication

Definition

Multiplication is an operation that is a quick way of doing repeated addition.

This operation is denoted with the \(\times \) symbol.

The result of multiplication is called the product.

Example

If you multiply \(3\) to \(4\), you get \(12\).

That is \(3 \times 4 = 3 + 3 + 3 + 3 = 12\).

Properties

  • Multiplication is commutative:

    • \(a \times b = b \times a\)

  • The identity of multiplication is 1:

    • \(n \times 1 = n\)

  • Any number multiplied by \(0\) equal \(0\):

    • \(n \times 0 = 0\)

  • The product of a positive number and a negative numbers is a negative number.

    • If \(a\) or \(b\) is negative and the other number is positive:

    \(a \times b \lt 0\)

  • The product of two negative numbers is a positive number.

    • If \(a\) and \(b\) are both negative:

    \(a \times b \gt 0\).
