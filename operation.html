Operation

Definition

An operation is a process that takes two numbers and return a third one.

Addition and subtraction

Addition is an operation that combine two numbers to make a third one.

Subtraction is an operation that remove a number to another number.

Adding or subtracting any number to 0 never change its result.

Multiplication and division

Multiplication is an operation that indicates how many times a number is added to itself.

Division is an operation that involve separating a number into equal parts.

Multiplying or dividing any number by 1 never change its result.

Multiplying any number by 0 always equal 0.

Exponentiation

Exponentiation is an operation that indicates how many times a number is multiplied to itself.

Unlike addition and multiplication, changing the order of numbers may change the result.

Exponentiating any number by 1 never change its result.

Exponentiating any number by 0 always equal 1.

Exponentiating 1 to any number always equal 1.
