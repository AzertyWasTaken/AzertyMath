Division

Definition

Division is an operation that involve separating a number into equal parts.

Division is denoted with the รท symbol.

Dividing an integer n by another integer that is not divisible by n result in a fraction.

An integer is divisible by another integer if its division have no remainder.

Properties

Division is not commutative: changing the order of its numbers may change the result.

Dividing any number by 1 never change its result.

Dividing any number (except 0) by itself equal 1.

Divisor

A divisor of an integer n is an integer that may be multiplied by some integer to produce exactly n.

The divisor count of the first n numbers are 1, 2, 2, 3, 2, 4, 2, 4, 3, 4.

Properties

Every integer is divisible by 1 and itself.

A number have a odd number of divisors if and only if it is square.

If an integer n is a divisor of an integer m, m / n is also a divisor of m.

Modulus

Modulus is an operation that associate to two numbers the remainder of their division.

Modulus is denoted with the % symbol.
